# braminus

Uses the [A* pathfinding algorithm](http://theory.stanford.edu/~amitp/GameProgramming/AStarComparison.html#the-a-star-algorithm)


### Development
  - `$ bundle exec rake` runs the tests
  - `$ bundle exec rackup` starts the web server
